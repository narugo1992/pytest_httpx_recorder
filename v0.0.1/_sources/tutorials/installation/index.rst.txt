Installation
===================

pytest_httpx_recorder is currently hosted on PyPI. It required python >= 3.8.

You can simply install pytest_httpx_recorder from PyPI with the following command:

.. code:: shell

    pip install pytest_httpx_recorder

You can also install with the newest version through GitHub:

.. code:: shell

    pip install -U git+https://github.com/narugo1992/pytest_httpx_recorder@main

script:
